Les fichiers `.mmd` utilisent la syntaxe **Mermaid** (compatible GitHub, GitLab, Notion, etc.).

## Rendu visuel

Pour visualiser un diagramme :
- **GitHub** : ouvrez le fichier `.mmd` directement (rendu natif)
- **VS Code** : installez l'extension *Markdown Preview Mermaid Support*
- **En ligne** : copiez le contenu sur [mermaid.live](https://mermaid.live)
- **Export PDF** : utilisez [mermaid-cli](https://github.com/mermaid-js/mermaid-cli)
